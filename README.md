Aplicación creada con React. 

Buscador de Pokemon por nombre y en base a las características de salud,ataque y defensa. 
Se muestra los pokemon pokemon en formato carta con los datos: 
- Nombre
- Tipo de pokemon
- Salud, Ataque y Defensa.
